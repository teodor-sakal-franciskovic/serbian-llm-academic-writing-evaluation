This repository is organized as follows:

- **analysed_results/**  
  Contains all analysed results produced during the experiments and mentioned in the paper's RQs. These include MATTER labeling results, summarised comparison results (LLM-generated vs human), and error pattern analysis.

- **llm_results/**  
  Contains all results generated by large language models (LLMs) - GPT-4o, GPT-5.2, Claude Sonnet 4.5, Gemini Flash 3.

- **scripts/**  
  Contains all scripts used to generate code and artifacts required for the research paper.

- **appendix/**  
  Contains prompts and rubrics used in the experiments, translated from Serbian to English. Also, contains an anonymised paper example written in Serbian language.

This structure is designed to support transparency, reproducibility, and ease of navigation for the research workflow.
